# Horseno - Next.js App Router

This project now uses Next.js with the App Router.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Open http://localhost:3000

## Build and Start

1. Build the project:
   `npm run build`
2. Start production server:
   `npm run start`
